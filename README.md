# Code comment

A gedit gdb plugin. Uses the side panel.

# Installation

Just follow these steps:

````
cd ~/.local/share/gedit/plugins
git clone https://github.com/CaF2/gedit-gdb-native.git
cd gedit-gdb-native
make
````

Then go into gedit -> settings -> plugins and enable this plugin

If it does not work, check that you have the gedit-devel package installed.
